A WordPress plugin that generates the post title based on the first 50 characters of the post content and includes ChatGPT functionality.

== Description ==

Generate Title & ChatGPT is a WordPress plugin that automatically generates post titles based on the first 50 characters of the post content. Additionally, it integrates ChatGPT functionality to assist users in generating content.

= Features =

- Automatic generation of post titles.
- Integration with ChatGPT for user assistance.

== Installation ==

1. Download the plugin ZIP file.
2. Log in to your WordPress admin dashboard.
3. Navigate to Plugins > Add New.
4. Click "Upload Plugin" and choose the ZIP file.
5. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

Once activated, the plugin will automatically generate post titles. To use ChatGPT functionality:

1. Edit a post or page in the WordPress editor.
2. Look for the "ChatGPT" section.
3. Enter your question in the provided input.
4. Click the "Submit" button.
5. The plugin will send your question to ChatGPT and display the response.

== Frequently Asked Questions ==

= How does the title generation work? =

The plugin extracts the first 50 characters of the post content and sets it as the post title.

= Can I disable ChatGPT integration? =

Yes, you can disable ChatGPT integration by modifying the plugin settings.
